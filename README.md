# Les Diablerets Panorama — versione finale

Sito statico trilingue (italiano, francese e inglese) pronto per GitHub Pages.

## File da caricare

Caricare nella radice del repository:

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`
- tutti i file immagine presenti nella cartella principale

Non modificare i nomi dei file e non spostare la cartella `images`.

## Immagini utilizzate

- Hero: Glacier 3000 e Peak Walk by Tissot
- Galleria: Glacier 3000, panorama invernale e villaggio alpino
- Inverno: panorama innevato
- Estate: massiccio e valle di Les Diablerets
- Contatto: panorama del villaggio

Tutte le immagini sono salvate localmente nel progetto: il sito non dipende da collegamenti fotografici esterni.

La fotografia estiva `summer-les-diablerets.jpg` deriva da Wikimedia Commons (`Diablerets0001.jpg` / Massif des Diablerets), con attribuzione indicata nella pagina Commons associata. Per una pubblicazione definitiva, conservare sempre i crediti e verificare le condizioni di licenza delle immagini scelte.

## Struttura semplificata
Tutte le immagini sono nella cartella principale, allo stesso livello di `index.html`. Non esiste alcuna cartella `images`.
